This package is responsible to control a mobile robot when receives a {\ttfamily go\+\_\+to\+\_\+point} command from the user.\hypertarget{md__r_e_a_d_m_e_autotoc_md4}{}\doxysection{Description of the system}\label{md__r_e_a_d_m_e_autotoc_md4}
In this R\+O\+S2 package, it is possible to communicate with nodes declared in our R\+O\+S1 package in order to command a mobile robot in a Gazebo environment. The behaviour of the robot will depend on commands given by the user, they can be\+:


\begin{DoxyItemize}
\item {\bfseries{start\+:}}
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item Receives a random value for X, Y and Theta
\item Based on the value of theta, rotates the robot until gets alligned with the goal value (X and Y)
\item Moves the robot towards the coodinates of X and Y
\item When reach the target, repeat the process
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item {\bfseries{stop\+:}} Because our user request is implemented as a service, the system will only read the user command when the robot reached its goal, being impossible to stop midway.
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md5}{}\doxysection{Codes and nodes}\label{md__r_e_a_d_m_e_autotoc_md5}
